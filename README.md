## Projeto Pixel Art

Implementação de uma página web que contém uma paleta de cores funcional que pode ser utilizada para criar desenhos em pixels.

## Habilidades

- Manipular o DOM

- Utilizar conhecimentos sobre JavaScript, HTML e CSS;
